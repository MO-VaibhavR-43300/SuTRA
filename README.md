# SuTRA

**SuTRA: Structurally-Unified Tokenization with Root Awareness**

Project page and resources for morphology-aware tokenization of Indic languages (Hindi, Marathi, Gujarati).

## Project page

Open [`index.html`](index.html) locally, or publish this repository with [GitHub Pages](https://docs.github.com/en/pages/getting-started-with-github-pages/configuring-a-pages-site-source-for-a-user-organization-or-project-repository):

1. Push this repo to GitHub.
2. **Settings → Pages → Build and deployment**: source **Deploy from a branch**, branch **`main`**, folder **`/ (root)`**.
3. Your site will be at `https://<username>.github.io/<repo>/`.

## Before publishing

1. **Figures** — export from the paper and save as:
   - `images/figure1_shattering.png` (Figure 1)
   - `images/figure2_overview.png` (Figure 2)
   - `images/figure3_qualitative.png` (Figure 3)
   - `images/sutra_poster.png` (optional poster for `poster.html`)

2. **Links** — in `index.html`, set:
   - `#paper-link` → arXiv or PDF URL
   - `#code-link` → GitHub repository URL

3. **BibTeX** — update venue and fields in `index.html` when the paper is accepted.
